# 要件定義書

## 目的

- ユーザー(自分)の体重管理
- 運動や体重記録を可視化し、習慣化させる

## 業務用件

| 管理番号 | 業務           | 説明                                 | 実施者   |
| -------- | -------------- | ------------------------------------ | -------- |
| FH-01    | 体重記録       | 一日に一度、体重を記録してもらう     | ユーザー |
| FH-02    | 体重入力の受付 | ユーザーごとのデータを DB に蓄積する | システム |

## 機能要件

| 管理番号 | 機能           | 説明                                                 | 分類 | 備考                   |
| -------- | -------------- | ---------------------------------------------------- | ---- | ---------------------- |
| FR-01    | TOP ページ     | システムについて説明する TOP ページを有する          | UI   | レスポンシブ対応を想定 |
| FR-02    | ダッシュボード | ユーザーの体重記録、毎日の記録の様子を表示するページ | UI   |                        |
